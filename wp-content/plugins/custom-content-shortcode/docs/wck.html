WCK Fields

For WCK fields, specify the metabox name first, then use the [field] shortcode.

Display WCK fields

[metabox name="metabox_name"]     [field field_name]     [field another_field] [/metabox]

Parameters

  • image - display an image field
  • size - size of image: thumbnail, medium, large, full (default) or custom defined size
  • shortcode- set true to enable shortcodes inside the field. Usually WCK pre-processes a text area to create break lines, but it doesn't work well with shortcodes. When shortcodes are enabled, you'll need to insert <br> tags manually.

Repeater

Use [repeater] to display a repeating metabox.

Display repeater fields

[repeater meta="metabox_name"]     [field field_name]     [field another_field] [/repeater]

Parameters

  • meta - name of metabox
  • id - post id (default is current post)

Inside loop

Display repeater fields from five recent posts

[loop type="post_type" count="5"]     Post Title: [field title]     [repeater meta="metabox_name"]         [field field_name]         [field another_field]     [/repeater] [/loop]
